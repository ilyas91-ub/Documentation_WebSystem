# Contribution — règles rapides

Merci de vouloir contribuer ! Voici quelques règles simples pour ce petit projet de documentation :

1. Fork ou clonage local
   - Clone ce dépôt et crée une branche pour ta modification :
     ```bash
     git checkout -b feat/ma-modif
     ```

2. Format et style
   - Édite `index.html` pour ajouter/modifier le contenu.
   - Utilise un style simple et propre (HTML sémantique : sections, titres, listes).
   - Si tu modifies le CSS, mets à jour `asset/css/index.css`.

3. Tests rapides
   - Ouvre `index.html` localement pour vérifier l'affichage.
   - Si possible, lance un serveur local (python -m http.server) et vérifie la navigation.

4. Pull Request
   - Pousse ta branche et crée une PR avec une description des changements.
   - Si le changement est majeur, explique le but et les régressions possibles.

5. Licence
   - En contribuant, tu acceptes que ta contribution soit intégrée sous la licence MIT du projet.

Merci !
